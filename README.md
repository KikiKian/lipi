# Lipi
Lipi is a Gujarathi phonetic system that converts english text into Gujarathi.

## Live Demo
[lipi.up.railway.app](https://lipi.up.railway.app)

## Controls
See [CONTROLS.md](CONTROLS.md) for the full mapping reference.

## Quick Example
| Type         | Gujarati |
|--------------|----------|
| ```k`em chh`o```   | કેમ છો   |
